# TASK 2 : ROCK-PAPER-SCISSORS GAME

# PROJECT DESCRIPTION :

A Python-based CLI (Command Line Interface) game where users can play Rock-Paper-Scissors against the computer. The program tracks scores and offers a seamless "play again" experience.

# PROJECT VIDEO :

https://github.com/user-attachments/assets/cf6f3bac-fb1c-431f-8606-add5fddc4493

# PROJECT FEATURES :

* User vs. Computer: Randomized computer moves using the `random` library.

* Score Tracking: Real-time tracking of user and computer points.

* Input Validation: Handles invalid user inputs (e.g., typing "lizard" instead of "rock").

* Continuous Play: Option to restart the game without exiting the program.

# TECHNICAL STACK :

* Language: Python 3.x

* Module: `random` (for computer move generation)

# HOW TO RUN :

1. Clone the repository.

2. Navigate to the `Task2_Rock_Paper_Scissors` folder.

3. Run `python rock_paper_scissors.py`.
