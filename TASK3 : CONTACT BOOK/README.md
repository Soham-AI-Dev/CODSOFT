# TASK 3 : CONTACT BOOK
# PROJECT DESCRIPTION :

A Python-based contact management system that allows users to store and manage contact information effectively. This project demonstrates skills in data handling, user input validation, and implementing a functional Command Line Interface (CLI).

# PROJECT VIDEO :

https://github.com/user-attachments/assets/3c1dcd09-cd6f-4075-bfc6-37ba8a5fa8b1

# PROJECT FEATURES :

* Add Contact: Store name, phone number, email, and address for each entry.

* View Contact List: Display a list of all saved contacts with names and phone numbers.

* Search Contact: Quickly find contacts using names or phone numbers.

* Update Contact: Modify existing contact details to keep information current.

* Delete Contact: Remove outdated or unnecessary entries from the book.

# TECHNICAL STACK :

* Language: Python 3.x

* Data Structure: Dictionary/List for in-memory storage (or JSON/CSV for persistence).

# HOW TO RUN :

1. Clone the repository.

2. Navigate to the `Task3_Contact_Book` folder.

3. Run `python contact_book.py`.
