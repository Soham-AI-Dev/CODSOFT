# TASK 1 : CALCULATOR
# PROJECT DESCRIPTION :
A GUI-based calculator developed using Python and the Tkinter library during my CodSoft internship. This project focuses on creating a clean user interface and handling basic arithmetic logic.

# PROJECT VIDEO :

https://github.com/user-attachments/assets/ed7bf3f6-f2dc-417d-8e77-ab61c5c9c66f

# PROJECT FEATURES :
* Error Handling: Manages division by zero and invalid inputs gracefully.

* Full Arithmetic Suite: Supports addition, subtraction, multiplication, and division.

# TECHNICAL STACK :
* Language: Python 3.x

* Library: Tkinter (Standard GUI Library)

# HOW TO RUN :
Clone the repository.

Navigate to the TASK1 : CALCULATOR folder.

Run python calculator.py.
